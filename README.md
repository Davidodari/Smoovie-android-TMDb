# Smoovie-android-TMDb
Android App Querying TMDb Using REST Libraries:Okhttp,Gson,Retrofit and Picasso for image loading

## PREREQUISITES

```
TMDb Api Key
```

An Api key is required to run GET Responses

Use in TMDB Class
